自助投币洗衣机系统设计

1、开机显示00，持续显示00，等待按键输入。

2、按键按一下，显示01，内部定时计数器开始计数。
	如果15秒内按键没有再次按下，则执行脱水操作。
        
3、如果15秒内按键再次按下，显示02，内部定时计数器开始计数。
	如果15秒内按键没有再次按下，则执行快洗操作。
        
4、如果15秒内按键再次按下，显示03，内部定时计数器开始计数。
	如果15秒内按键没有再次按下，则执行标准洗操作。
        
5、如果15秒内按键再次按下，显示04，立即执行洗大件。

这里使用按键key模拟投币动作，led点亮表示电机开始工作。利用FPGA做了一个简易的小模型。
